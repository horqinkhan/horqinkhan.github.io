lesson1 课程介绍

第一节课主要介绍了jupyter的操作方法和一些简单的python方法

  • 由于教授东一脚西一脚的讲课方法，所以笔记会很杂乱，希望怀着学习目的来阅读这份博客的朋友见谅

  1. sep=() 默认是空格，表示两个字符串之间用什么分割。 

    例 t1 = 'hi guys!'
   t2 = 'i am hero!'
   t3 = 'i am flashman!'
   print(t1, t2, t3, sep=',')
结果为 hi guys!,i am hero!,i am flashman!

  2. istitle() 检测字符串中所有的单词拼写首字母是否为大写，且其他字母为小写

    例 'Is'.istitle()
结果为 Ture

  3. lower() 转换字符串中所有大写字符为小写

    例 'Conversion TO loweCASE'.lower()
结果为 'conversion to lowecase'

  4. dir() 是一个内置函数，用于列出对象的所有属性及方法

  5. type() 可以用于判断一个变量的类型

    例 type('Conversion TO loweCASE')
结果为 str

  6. join() 将序列中的元素以指定的字符连接生成一个新的字符串

    例 L = ['111', '222', '333']
   '加上'.join(L)
结果是 '111加上222加上333'

  8. while true， try， raise的结合用法

    # 通过捕获错误的交互数据来提示输入错误
while True:
    try:
        user_input = int(input('Give a number between 10 and 20 included:'))
        if 10 <= user_input <=20:
            print(user_input ** 2)
            break
        else:
            raise ValueError# 表示引发该类型异常
    except ValueError:
        print('That is not a valid input, try again')

# 代码简化
while True:
    try:
        user_input = int(input('Give a number between 10 and 20 included:'))
        if not (10 <= user_input <=20):
            raise ValueError
        print(user_input ** 2)
        break
    except ValueError:
        print('That is not a valid input, try again')

    # 模拟10000000次抓取，计算字典中三个元素的取出概率

counts = {'A': 0, 'B': 0, 'C': 0}
N = 10_000_000
for i in range(N):

for k in counts:

  4. 求两因子与其最大公因数的倍数

    from math import gcd#导包
def simplify(numerator, denominator):
    the_gcd = gcd(numerator, denominator)#给the_gcd变量赋值
    return numerator // the_gcd, denominator // the_gcd#返回结果
simplify(234,3456)#结果为：(13, 192)

#处理字符串得到结果的方法
simplify(*convert_str_to_int('234','3456'))

  7. sorted()

    1. sorted()函数：从可迭代的项中返回一个新的排序列表,简而言之就是排序

      sorted({0,3,19,3452,81})

输出结果：

      D = {1: 10, 0: 20, 3: 30}

    3. 使用key来限定排序规则

      sorted({'Integer', 'literals', 'in', 'Different', 'bases'})

输出结果：
['Different', 'Integer', 'bases', 'in', 'literals']
#这里可以看到大写字母都排在前面
#如果列表中的元素是字符串，那么key = str.lower可以使比较都按小写字母的ASCII码进行比较

sorted({'Integer', 'literals', 'in', 'Different', 'bases'},
          key = str.lower #.lower()函数效果一样
          )

输出结果：
['bases', 'Different', 'in', 'Integer', 'literals']

    4. 如果希望既按字符串长度进行排序，又按ASCII码排序，可以定义一个函数

      def combine_orders(s):
        return len(s), s
sorted({'Integer', 'literals', 'in', 'Different', 'bases'},
              key = combine_orders
              )

输出结果：
'in', 'bases', 'Integer', 'literals', 'Different']
