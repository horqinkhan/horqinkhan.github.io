lesson1 课程介绍

第一节课主要介绍了jupyter的操作方法和一些简单的python方法

  • 由于教授东一脚西一脚的讲课方法，所以笔记会很杂乱，希望怀着学习目的来阅读这份博客的朋友见谅

  1. sep=() 默认是空格，表示两个字符串之间用什么分割。 

    例 t1 = 'hi guys!'
   t2 = 'i am hero!'
   t3 = 'i am flashman!'
   print(t1, t2, t3, sep=',')
结果为 hi guys!,i am hero!,i am flashman!

  2. istitle() 检测字符串中所有的单词拼写首字母是否为大写，且其他字母为小写

    例 'Is'.istitle()
结果为 Ture

  3. lower() 转换字符串中所有大写字符为小写

    例 'Conversion TO loweCASE'.lower()
结果为 'conversion to lowecase'

  4. dir() 是一个内置函数，用于列出对象的所有属性及方法

  5. type() 可以用于判断一个变量的类型

    例 type('Conversion TO loweCASE')
结果为 str

  6. join() 将序列中的元素以指定的字符连接生成一个新的字符串

    例 L = ['111', '222', '333']
   '加上'.join(L)
结果是 '111加上222加上333'

Lesson2 第二节课讲了数据类型和一些相关函数，以及简单涉及了循环

  1. 数据类型-字典(dictionary)

    multiplication_table = {(2,3): 6,(4,7): 28, (5,5): 25}
# 用key得到对应的字典中的值
multiplication_table[(2,3)] # multiplication_table[2,3] 也 ok
# 加一个元素在字典中
multiplication_table[6,2] = 12

  2. 列表

    L = [2, 4, 1, 7]
# 取首元素
print(L[0])
# 取列表长度
print(len(L))
# 取最后一个元素
print(L[len(L)-1])
# 取从右往左数第四个元素
print(L[-4])
# 把第二个元素的值改为40
L[1] = 40
# 加一个（100）的元素在列表最后
print(L)
L.append(100)

  3. 元组

    L1 = [1, 2]
L2 = [2, 2]
L3 = [3, 4]
T = L1, L2, L3
print(T)
# 元组是只读的，所以无法直接修改，如果想要更改元组内容，可以通过修改元组内元素
# 如 L4 = [2, 20], T[1] = L4
# 或 T[1] = 20 都是行不通的
L2[1] = 20

  4. 列表可以直接运算

    L = [20, 321, 1, 8122]
[str(x) for x in L]# 将列表内元素更改为字符串
[2 * x + 1 for x in L]

  5. .split()函数

    s = 'My    NAme \t 12 1is Silas'
# 通过指定分隔符对字符串进行切片, 默认为以空格为分割进行切片
s.split()
结果：
['My', 'NAme', '12', '1is', 'Silas']

# .split(分隔符，num)如果参数 num 有指定值，则分隔 num+1 个子字符串
s.split(' ', 2)
结果：
['My', '', '  NAme \t 12 1is Silas']

  6. .lower()

    # 大写变小写
s.lower()
结果：
'my    name \t 12 1is silas'

  7. .startswith() and .endswith

    s.startswith('M')
# 检查字符串是否以指定值开头，返回True和False
s.endswith('s')
# 判断字符串是否以指定后缀结尾,如果以指定后缀结尾返回True,否则返回False

  8. .join()

    T = 'one', 'two', 'three'
'_*_'.join(T)
# 将序列中的元素以指定的字符连接生成一个新的字符串
结果：
'one_*_two_*_three'

  9. 一个小循环

    L = [20, 11, 53, 79, 804]
for e in L:
    oneoneone = '1','3','5','7','9'
    for one in oneoneone:
        if str(e).endswith(one):
            print(e)
# 将大循环体中的e不断代入小循环one中，判断L中每个元素的结尾是否属于oneoneone

# import：导入模块、文件、包

  2. Randrange

    print('100到1000间一个随机的2的倍数:',random.randrange(100,1000,2))
# randrange 语法
# random.randrange(start, stop, step)区间左闭右合

  3. 还是randrange

  4. 简单的for循环

    L = [0, 2, 1, 7, 2, 4]
for e in L:
        if e >= 4:
            continue # 跳过当前循环的剩余语句，然后继续进行下一轮循环
        #pass 占位语句，不做任何操作
        #break 跳出整个循环
        print(e)

  5. 简单的while循环

    L = [0, 2, 1, 7, 2, 4]
not_too_large = True
i = 0
while i < len(L) and not_too_large:
    e = L[i]
    if e >= 6:
        not_too_large = False
    else:
        print(e)
        i += 1

  6. if和elif的区别

    # elif：如果条件1判断成功的话，条件2就不会继续判断了
# if：互相独立
L = [0, 2, 1, 7, 2, 4, 0, 12, 5, 20, 14, 6]
for e in L:
    if e < 2:
        print(e)
    elif e < 5:
        print(10 * e)
    elif e < 10:
        print(100 * e)
    else:
        print('Hi')

  7. Try的用法

    # 捕获异常
L = {}
try:
    L[0] # 尝试执行的代码 
    print('GOOD')
except (KeyError, IndexError): # 出现异常的处理
    print('BAD')

  8. while true， try， raise的结合用法

    # 通过捕获错误的交互数据来提示输入错误
while True:
    try:
        user_input = int(input('Give a number between 10 and 20 included:'))
        if 10 <= user_input <=20:
            print(user_input ** 2)
            break
        else:
            raise ValueError# 表示引发该类型异常
    except ValueError:
        print('That is not a valid input, try again')

# 代码简化
while True:
    try:
        user_input = int(input('Give a number between 10 and 20 included:'))
        if not (10 <= user_input <=20):
            raise ValueError
        print(user_input ** 2)
        break
    except ValueError:
        print('That is not a valid input, try again')

  9. 一个概率计算代码

    # 模拟10000000次抓取，计算字典中三个元素的取出概率
from random import choice
L = 'A','B','C'
counts = {'A': 0, 'B': 0, 'C': 0}
N = 10_000_000
for i in range(N):
    counts[choice(L)] += 1
for k in counts:
    print(k, 'has been choisen', counts[k] / N * 100, 
          '% of the time',sep=''
         )

    #可以加一些更改让输出语句变得更易阅读，如最后一个打印可改为
    #print(k, f'has been choisen {counts[k] / N * 100:.2f}%', 
              'of the time',sep=''# ':.2f'--取小数点后两位
             )
