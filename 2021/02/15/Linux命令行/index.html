Basic Linux Commands

发表于 2021-02-15 更新于 2021-02-16

lab中linux操作系统的基础：

以下先记录一些常用的linux commands

  1. “pwd” 指 “print working directory”, 也就是打印当前路径
    • “cd ../“ cd指 “change directory”, 切换当前目录至父目录
    • “cd ./“, 切换至当前目录
    • “cd ~” 指 “home directory”
    • “cd /“ /指 “absolute path”, 切换至指定路径
    • “ls” 指 “list directory”, 查看当前目录
    • “ls -l” 指 “long listing”, 以长数据串的形式列出当前目录
    • “ls -a” 指 “show all the hiden files and current dirctory”, 列出全部文件(包括.开头的隐藏文件)
    • “ls -la” 可获得更多detail
    • “cp” 指 “copy”, 复制操作
    • “cp -r” 指 “recursive copy”, 递归式复制，用于目录的复制
  5. “rm” 指 “remove”, 删除操作
  6. “cat” 指 “display the content”, 查看文件内容
  7. “mv” 指 “move files and directories”, 移动文件
  8. “ssh” 指 “securely connect to remote server”,远程服务

作为操作Linux系统的基础，熟练掌握命令行在初学阶段显得尤为重要，可以多做练习。同时随着学习，更多的操作命令也会在之后更新。
